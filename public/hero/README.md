# ヒーロー画像の差し替え

このフォルダの画像を差し替えると、トップのヒーロースライドの写真が変わります。

- **hero-1.svg** → 1枚目のスライド（5秒ごとに切り替わります）
- **hero-2.svg** → 2枚目
- **hero-3.svg** → 3枚目

## 差し替え方法

1. 同じファイル名（hero-1, hero-2, hero-3）で、ご自身の写真（.jpg / .png 推奨）を用意する
2. このフォルダに配置する（例: `hero-1.jpg`）
3. `components/Hero.tsx` の `HERO_IMAGES` の拡張子を変更する  
  例: `"/hero/hero-1.svg"` → `"/hero/hero-1.jpg"`

## 推奨サイズ

- 横 1920px × 縦 1080px 程度（16:9）
- または同じアスペクト比であれば自動でトリミングされます
